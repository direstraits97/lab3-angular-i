# Laboration 3 - Angular

Detta repo innehåller de filer som skapat en webbplats med hjälp av ramverket Angular. Webbplatsen följer en Single Page Application-struktur, alltså att en initial laddning görs och endast nytt innehåll laddas istället för en hel html-sida vid varje klick.

Webbplatsen har tre undersidor: en startsida, en sida som konverterar värden samt en sida som beskriver arbetet med slutsatser om processen. För att optimera innehållet med kortare laddningstid har komponenter för sidhuvud och sidfot skapats, och med hjälp av Router-modulen slipper redan laddat innehåll laddas om.

Den huvudsakliga koden som skapats används under sidan som konverterar värden. Med hjälp av databindning kunde properties från klassen skrivas ut direkt i HTML-filen, efter att de genomgått de metoder som anropats.
